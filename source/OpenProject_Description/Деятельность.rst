Деятельность
+++++++++++++

.. image:: /OpenProject_Description/pics/main_page_activity.png

Данный раздел посвящен всем изменениям, производимым в программе с момента 
работы в ней до настоящего времени. Слева на панели имеются фильтры, по которым
можно задать параметры отображения деятельности, установив галочку в 
соответствующий квадрат.
